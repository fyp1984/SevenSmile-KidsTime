## 4. 重点验证说明

### 4.1 表单采集

- 五类成员都能进入统一邀请流程
- 联系信息最少需填写一种
- 康复人员额外要求填写 `康复进度记录`
- 表单字段覆盖：
  - 基础身份信息
  - 与主账号关系
  - 联系方式
  - 健康档案
  - 照料需求说明

### 4.2 邀请时效

- 支持 `24 小时 / 72 小时 / 7 天`
- 生成邀请时同步写入：
  - `link`
  - `qrText`
  - `verifyCode`
  - `expiresAt`
  - `expiresLabel`
- 支持主动标记过期与撤回

### 4.3 权限配置

- `管理员 / 协作者 / 执行者 / 只读 / 宝贝` 已映射到不同权限摘要
- 成员卡与编辑页同步刷新权限结果
- 家长身份才可执行成员与邀请相关操作

### 4.4 敏感信息保护

- `phone / wechat / email / idNo` 以加密字符串保存
- 列表页显示 `maskPhone / maskWechat / maskEmail / maskIdNo`
- 编辑页需要显式进入成员编辑抽屉才可查看回填值
